# 🌐 Visual Sitemap Generator

A premium, high-performance visual sitemap generator that transforms any website into an interactive, hierarchical D3.js tree. 

![Dotted Background Pattern](https://visual-sitemap.vercel.app/preview-placeholder.png) <!-- Link to a screenshot if you have one public -->

## 🚀 Live Demo
**[visual-sitemap-v3.vercel.app](https://visual-sitemap-v3.vercel.app)**

## ✨ Features
- **Real-time Crawling**: Dynamic sitemap generation from any URL via Vercel Serverless Functions.
- **Interactive D3.js Tree**: Fluid animations, zoom, and pan controls.
- **Bi-directional Sync**: Seamless interaction between the map and the hierarchical sidebar.
- **SEO Insights**: Metadata extraction (Title, Description, Keywords) and page previews.
- **Premium UI**: Modern dark theme with radial-gradient dotted background and smooth micro-animations.

## 🛠️ Tech Stack
- **Frontend**: Vanilla JS, D3.js (Visualization), CSS3 (Modern Glassmorphism).
- **Backend**: Node.js, Axios, Cheerio (Serverless Crawling).
- **Hosting**: Vercel (Speed & Global Edge).

## ⚙️ Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/nestorwwh-hue/visual-sitemap.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the local server:
   ```bash
   node server.js
   ```
